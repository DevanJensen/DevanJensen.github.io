JavaScript Arrays

An array is a collection of data.

When we write programs, we often need to work with collections of data. For example, you probably wouldn't write a program to keep track of a single book, instead it would be more useful to keep track of a collection of books (maybe you are building software for a public library).

JavaScript, like many other programming languages, allows us to work with collections of data by using arrays.

In previous lessons, we have used variables to store a single piece of data, such as a number, string, or boolean. (although you might argue that we used variables to contain objects that can store multiple properties, these variables still referred to a single object). Arrays allow us to use a single variable to store multiple values, consider this example: 

var myTestScores = [96, 85, 89, 77];

This is an array of numbers. Let's take a minute to discuss the syntax used to create this array. Notice that there are square brackets enclosing the array []. And notice that the items in the array are separated by commas.

The previous code sample demonstrates and array that stores numbers. Here's one that stores strings: 

var myFriends = ["Joe", "Betty", "Sally"];
console.log(myFriends);

Some languages require that arrays store values that are the same data type. But JavaScript allows you to mix data types within an array. 

var mixedArray = [6, "bob", false];

Array Indices

The values stored in an array are called elements (not to be confused with elements in HTML). The mixed array above stores 3 elements.

Each element in an array has a specific position, which is known as the index of the element. For example, in the mixed array above, the first element (the value 6) has an index of 0 Array indexes start at 0 in most programming languages, this is something that can trip up beginners, but with enough practice, you'll quickly get used to this.

You can retreive values out of an array by using it's index with bracket notation. 

var myTestScores = [96, 85, 89, 77];
var firstTest = myTestScores[0];
var secondTest = myTestScores[1];
var thirdTest = myTestScores[2];
var fourthTest = myTestScores[3];

Again, be careful to remember that the first element has an index of 0.

We can also change values in an array by using bracket notation and an index: 

var myTestScores = [96, 85, 89, 77];
console.log(myTestScores);

myTestScores[3] = 100;
console.log(myTestScores);

It's common to start with an empty array and build it up as the program runs. Notice in the code sample below, that we start with an empty array, and then we use indexes and bracket notation to add elements to the array. 

var testScores = [];
testScores[0] = prompt("Enter your first test score.");
testScores[1] = prompt("Enter your second test score.");
testScores[2] = prompt("Enter your third test score.");
console.log(testScores);

Arrays are Objects

Arrays are actually a special type of object, and like other objects, they have properties and methods that we can use. We'll talk more about array methods later, but for now let's go over the length property of an array. The length property will tell us how many elements are in an array. Note that the length will always be one greater than the highest index in an array. This makes sense when you remember that the index of the first element is 0. 

var myTestScores = [96, 85, 89, 77];
console.log(myTestScores.length);

You can use the length property when you want to add an element to an array. 

var myFriends = ["Betty", "Sally", "Bob"];
myFriends[myFriends.length] = "Pat";

In class we can discuss why this approach can be better than using an index number to add an element to an array. Likewise, we can always get the last element from an array like this: 

var myFriends = ["Betty", "Sally", "Bob"];
var myLastFriend = myFriends[myFriends.length - 1];
console.log(myLastFriend);

In the previous two examples we've put expressions in between the brackets rather than values. These expressions will be evaluated and the result will be used as the index of the array to fetch. For example, the expression myFriends.length - 1 evaluates to 2. So myLastFriend will contain the value "Bob".

The point is that you can put expressions inside the brackets, and they will be evaluated to an index number. The is a very powerful feature of arrays, as we'll soon see. Can you figure out what the following program does? 

var myFriends = ["Betty", "Sally", "Bob"];

var x = prompt("Enter a valid index number for the myFriends array");
var selectedElement = myFriends[x];

if(selectedElement){
  console.log("The value of the element at index " + x + " is: " + selectedElement);
}else{
  console.log("There is no element at index " + x + " in the myFriends array.");
}

There are different syntaxes that you can use when working with arrays. We'll go over some of them later, when we talk about array methods. But for now I'll just show you another way that you could create an array, because you may see this when you look at other people's code 

var numberArray = new Array(1,3,5,7);

// this is the same as... 
var numberArray = [1,3,5,7];

Arrays of Objects

We have seen that you can put various data types inside of arrays, such as numbers, strings, and booleans. Arrays can also store objects. This allows us to model collections of real world objects, which is very useful for processing data in our programs. Take a look at this array: 

var books = [];
books[0] = {title:"Harvey Porter", author:"Rowlings"};
books[1] = {title:"The Sun Also Rises", author:"Hemmingway"};

We start out with an empty array, and then we add an object at the first position (index = 0). Then we add another object at the second position (index = 1);

This approach required us to write three statements to create and populate the array. But we could do it all in a single statement like this (note that I broke the statement up into separate lines just to make it easier to read). 

var books = [
  {title:"Harvey Porter", author:"Rowlings"},
  {title:"The Sun Also Rises", author:"Hemmingway"}
];

If you wanted to drill down into this array, and for example, log out the title of the first book, you could do it like this: 

console.log(books[0].title);

// we could also change the title of the first book like this...
books[0].title = "Henry Popper";

Another common technique is to pull an item out of an array and store it in a variable, like this: 

var b = books[0]; 
console.log(b.title);

b.title = "Harry Potter";
console.log(books);

We shall soon see that you can write some very powerful programs when you start using arrays of objects!

Questions - JavaScript Arrays

Question 1

What is the term for items that are stored in an array?

Items inside of an array are called elements.

Question 2

The position number of an item in an array is called what?

It is known as the index.

Question 3

What is the index of the first value in an array?

The first index value in an array is always zero.

Question 4

True or False - The length of an array is always one greater than it's highest index.

False, it is always one less.
