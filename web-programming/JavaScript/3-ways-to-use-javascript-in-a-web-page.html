3 ways to use JavaScript in a web page.

  1. You can put JavaScript code inside a <script> </script> tag.
  2. You can use a <script> </script> tag to 'link' to a .js file that has JavaScript code in it.
    In order to do this, you must add a src attribute to the tag and set the value of the attribute to the .js file, for example:
    <script src="path/to/somefile.js"> </script>
    This is the recommended method because:
    • It allows you to keep your JS code separate from your HTML code
    • It allows you to use the same JavaScript code in many web pages
    Note that you usually put a script tag inside the head tag of a page, but you can put it in other places too.
  3. You can put JavaScript commands in HTML tag attributes.
    This is NOT recommended because:
    • Your code gets messy if it's sprinkled all over a web page in the attributes of tags.
    • It can become very difficult to maintain messy code.

Putting JavaScript Code Inside a Script Tag

  1. Add a 'script' tag inside the head tag of this page (make sure you have a start tag and an end tag).
  2. Copy this JavaScript code and paste inside the script tag: 
    alert('Hello World!');
  3. Reload the page to run the JavaScript code. Note that the alert appears before you see any of the content on the page
  4. Copy the entire script tag (including the start tag, end tag, and the JavaScript code in between) and paste it at the bottom of this page (after the closing html tag) Then reload the page and note that the second alert box appears after the content of the page is displayed.

Putting JavaScript Code Inside a .js File

  1. Create a file called test.js inside this folder (web-programming/JavaScript)
  2. Copy this JavaScript code and paste it inside of test.js (then save the file) : 
    alert('Hello from test.js');
  3. Add another script tag inside the head tag (include a start tag and end tag). Then add a src attribute to the script tag and set its value to test.js.
  4. Now reload this page to run the JavaScript code.
