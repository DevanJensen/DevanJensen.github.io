JavaScript variables that hold HTML elements

We previously talked about variables and data types in JavaScript. So you should know what a variable is, and you should know that variables can 'hold' different data types, such as numbers, strings, booleans, etc.. You should also know what it means to declare and initialize a variable.

You can also use variables to hold HTML elements that are on your page (remember that 'element' and 'tag' are interchangeable terms). This is one of the most useful things you can do in JavaScript, and it allows you to manipulate the HTML tags/elements on your page. When you have a variable that holds an HTML element, we often say that you have a 'handle' on the element. The variable itself is the handle, and we can use it to manipulate the HTML element.

There are various ways in JavaScript to 'get a handle' on an element, but a common approach is to use the id attribute of the element. If you look at the HTML code for this page, you'll see that on line 14 the H1 tag has an id attribute set to 'mainH1'.

Here is one way to get a handle on that H1 tag by referring to the value of its id attribute: 

var myElement = document.getElementById("mainH1");

At this point, don't worry about the details of he code on the right side of the equals sign, just understand that it initializes the variable myElement to hold the H1 tag in this page. And understand that we are able to get a handle on that specific H1 tag by referring to the value of its id attribute (which is 'mainH1'). When you have a variable that holds an HTML element, you can start to manipulate that element with your JavaScript code.
