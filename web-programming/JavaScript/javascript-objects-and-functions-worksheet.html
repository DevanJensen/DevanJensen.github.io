Objects and Functions

Objects as Parameters

Just like other data types, objects can be used as parameters when invoking functions.

In the following example we will create an object that represents a students record for a class (suppose that we are writing a program to keep track of grades in a class, the final grade depends on only two exams, the mid term exam and the final exam). 

var student1 = {name: "Sally", midTermExam:85, finalExam: 92};

Next we create a function that can calculate a student's average grade: 

function calculateStudentAverage(studentObj){
    var avg = (studentObj.midTermExam + studentObj.finalExam)/2;
    return avg;
}

Finally, we can invoke the function and pass in student1 as a paramter. 

var student1Avg = calculateStudentAverage(student1);
console.log(student1Avg);

In this example, the function expects an object as a parameter, and it uses the midTermExam and the finalExam properties of the object to calculate the student's average score for the class. So the function not only expects an object as a parameter, it also expects that object to have properties named midTermExam and finalExam. If we made a mistake, and used an object that doesn't have a midTermExam or finalExam property, we would find that our program will not work correctly. We could address this by adding code that checks the object parameter, but we'll save that for later.

This example program is pretty crude, but it demonstrates a powerful pattern that is often used in programming. Remember that most programs are written in order to work with data, and that we can create objects to represent the data we need to work with. In this example we've seen how you can use objects and function to process data in your programs.

Objects as Return Values

We've already learned that functions may have return values. Those return values can be any data type, including objects.

Let's assume that we are creating a program that works with graphics, and we need to draw a bunch of rectangles. Let's also assume that we need to change the size of rectangles, so we write a function that takes a rectangle object as parameter, and another parameter that indicates scale the new rectangle. The function returns a new rectangle object whose size will be based on the parameters. 

function makeResizedRect(rect, multiplier){

    var resizedRect = {
        width: rect.width * multiplier,
        height: rect.height * multiplier
    };

    return resizedRect;
}

// create an object that represents a rectangle...
var myRect = {width:20, height:50};

// use myRect as a parameter to call makeResizedRect()
var newRect = makeResizedRect(myRect, 2);
console.log(newRect);
