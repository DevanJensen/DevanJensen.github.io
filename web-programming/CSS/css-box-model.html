Setting the Height and Width of an Element

Block-level elements can have a height an width set. If you try to set the height and width of an inline element, they will be ignored. 

height: 200px;
width: 300px;

Setting the Border of an Element

To set the border around an element you must specify the width, color, and style (examples of border styles are solid lines, dotted lines, dashed lines, etc.) 

border-width: 2px;
border-color: red;
border-style: solid;

You can also set the border for just one side of the element like this: 

border-top-width: 5px;

There is a shorthand approach to setting the border that allows you to specify the width, color, and style in a single rule: 

border: 1px dashed yellow;

Setting the Margin and Padding of an Element

The margin is the space around the outside of an element. Adding margin gives your elements breathing room. 

margin-top: 30px;
margin-right: 30px;
margin-bottom: 30px;
margin-left: 30px;

Padding is the space around the content of an element. Padding appears inside the border of an element, while margin is outside of the border. 

padding-top: 12px;
padding-right: 12px;
padding-bottom: 12px;
padding-left: 12px;

If you want to apply the same margin and/or padding to all sides of an element you can do so like this: 

margin: 10px;
padding: 5px;

There is also a short hand way of applying different margin/padding each side of an element. If you specify two values, the first will be applied to the top and bottom sides, and the second will be applied to the left and right sides. If you specify four values, they will be applied top, right, bottom, and left sides (start at the top and move clockwise). 

/*set the top and bottom to 10px and the left and right to 30px*/
margin: 10px 30px; 

/*set the top to 10px, the right to 20px, the bottom to 30px, and the left to 40px*/
padding: 10px 20px 30px 40px;

Here's a trick that you can use to center a block-level element inside of it's parent, just set the margin-left and margin-right to 'auto' and the browser will calculate the margin to apply based on the width of the element. 

margin: 0 auto;

What is the true width and height of an element?

Note that if you set the width of the an element, and then add a border, padding, and margin, the true width will be the combination of all of them. So you must be careful to account for padding, border, and margin when calculating the dimensions of your elements within a page.

Using % as a unit of measure instead of pixels

You can specify height and width in percent rather than using pixels. The actual width/height of an element will be calculated based on the width/height of it's parent element. This can be very helpful in creating a fluid layout, in which the web page adapts itself to various screen sizes. 

width: 50%;

Changing an element's 'display' property

You already know that in HTML there are block-level and inline tags. You can set the display property in CSS to change this. The display property may be set to one of the following values:

  • block - This will make an element block-level
  • inline - This will make an element inline
  • none - This will hide an element
  • inline-block - The element will behave like an inline element, but will allow you to set it's height and width

Making images 'fluid'

Here is a great trick to make your images adapt to various screen sizes. Put your images inside a div and apply a class to the div so that you can target it with your CSS selectors without targeting all divs. Apply a width to the divs (this will be the 'standard' size of images in your page). Then put your images inside these divs. Finally, set all image tags that are inside the div tags to a width of 100%. 

.img-container{
	width:90%;
    margin: 0 auto;
}
.img-container img{
	width: 100%;
}
