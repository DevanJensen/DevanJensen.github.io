HTML Odds and Ends

What is the purpose of a div tag?

The div tag defines a section of an HTML document, usually used to group up different elements so they can be styled by CSS.

Is a span tag inline or block-level?

The span element is inline.

When would you want to use a pre tag in your code?

You would want to use it when you want spaces to show up in the text.

What is an entity character?

Entity characters start with ampersand(&) and are used when one wants to display text that would normally be considered an HTML code, such as <h1> Hello World! </h1>.
