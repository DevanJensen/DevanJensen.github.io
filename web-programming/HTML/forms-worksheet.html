Forms

In the form tag near the bottom of the page, add an HTML table that has two columns and 6 rows. The first column will contain a label and the second will contain an HTML form control.

Click here to see what the finished form and table will look like.

Here are the details for each row in the table. 

LABEL            HTML Form Control (in the second column of the table)
----------------------------------------------------------------------------------------------------------------------
First Name:      Create a text input that has a name attribute of 'txtFirstName'.
                 The input should also have an id attribute of 'txtFirstName'.

Favorite Color:  Create a select box that has a name attribute of 'selColor'.
                 The select box should also have an id attribute of 'selColor'.
                 The select box should contain the following options:
                 - Red (the value attribute of this option should be set to 'red').
                 - Yellow (the value attribute of this option should be set to 'yellow' ).
                 - Blue (the value attribute of this option should be set to 'blue').
                 - Other (the value attribute of this option should be set to 'other').
                 
                 For example: the first option inside the select tag should look like this:

                 <option value="red">Red</option>

Favorite Meal:   Create a group of radio buttons (put a label to the right of each radio button)
                 - Breakfast (give it a name attribute of 'rgMeal', a value attribute of 'breakfast', and an id attribute of 'rbBreakfast'). 
                 - Lunch (give it a name attribute of 'rgMeal', a value attribute of 'lunch', and an id attribute of 'rbLunch').
                 - Dinner (give it a name attribute of 'rgMeal', a value attribute of 'dinner', and an id attribute of 'rbDinner').

Places Visited:  Create 3 checkboxes with a label to the right of each one (put a line break after each label)
                 - Europe (give it a name attribute of 'chkEurope', an id attribute of 'chkEurope', and a value attribute of 'Europe').
                 - Africa (give it a name attribute of 'chkAfrica', an id attribute of 'chkAfrica', and a value attribute of 'Africa').
                 - Asia (give it a name attribute of 'chkAsia', an id attribute of 'chkAsia', and a value attribute of 'Asia').

Favorite Quote:  Create a textarea that has a name attribute of 'txtQuote' and an id attribute of 'txtQuote'.       

Submit           Create a submit button (an input tag with a type attribute of 'submit' and a value of 'Submit').
                 You can omit the label in the first column of this row in the table.
