Why would we want to open a link in a new browser tab?

So that the user remains on our webpage if the link takes them to another website.

What is 'bread crumb navigation'? And what are some advantages that it offers to users of a web site?

It tells the user where they currently are in the webpage. It can help the user navigate the site as they know their current location on it.
