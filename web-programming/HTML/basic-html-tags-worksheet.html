Basic HTML Tags

What is the significance of the html element in an html document?

It is the element that contains all other elements on the page. It is the root element.

What is the significance of the head element in an html document?

It's the element that contains the title, any scripts it needs, styles for the pages, metadata information and much more.

What is the significance of the title element in an html document?

It supplies the name of the webpage to the browser's tab.

What is the significance of the body element in an html document?

Everything inside the body element is placed on the webpage its self.

What is the difference between an inline element and a block-level element?

Inline elements, unlike block-level elements, does not create a new line above or below the element. The bold element is an example of an inline element, while the h1 element is an example of a block-level element.

What is a self-closing tag?

This classifies when an element, such as the Body element, ends.
