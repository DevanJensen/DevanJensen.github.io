Why should you set the width and height of an image even if you don't have to?

The website will know how much space it takes up before the image loads.

Why is it important to use small file sizes for images (image files that are small in kilobytes)?

It helps the image load faster on slower connections such as phone data.

What types of images should be saved as .jpg files, and what types of images should be saved as .png files?

Using .jpg is good for smaller file sizes and images with more color pop at the sacrifice of quality, while .png is good for images that require transparency and/or higher quality.

Most web browsers display images at what resolution (how many ppi)?

Most web browsers display images at 72ppi, otherwise images will look pixelated on most displays.

If you want an image to have a transparent background, what type of image should it be (what file extension would you use)?

.png images are for inserting transparent images.

What are responsive images?

Responsive images automatically adjust to the size of the display.
