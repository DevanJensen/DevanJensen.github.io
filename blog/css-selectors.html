Using CSS Selectors


To use CSS, you must put the code inside of a style tag or inside of a seperate .css file that the page links to. 

	To select an h4 tag, you must put h4 {}, and then put code within the brackets. You can also select code by id if you use #[IdName].
	You can also apply different kinds of code such as :hover so that the text only changes if your mouse pointer hovers over the text.
	There are muitple different lines of code you can use, here are a few examples:
					
		margin:; = Applies a margin around the tag, you can apply the margin to only left, right, top, or bottom by putting -[insert position here] at the end. Example: margin-left:;

		border:; = Applies a border around the text, the size, type, and color can be selected. Example: border:10px dotted purple;

		color:; = Applies a color to the text, accepts typing out color names, using the RGB color spectrum, or hex color codes. Example: color: red;

		background-color:; = Applies a background color, runs off the same rules as color:; Example: background-color: orange;

		padding:; = This paddens the text from it's own boundaries, it's easily visible if you have a
		border set or background-color as those reveal the boundary box of the element.

		font-size:; = Changes the size of text. Example: font-size: 50px;

		font-family:; = Changes the font type. Example= font-family: Times New Roman;

					
	
	code example:

	h4 {
	font-size: 15px;
	font-family: comic Sans MS;
	padding: 10px;
	margin-left:100px
	}
