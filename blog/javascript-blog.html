Using Javascript


	Javascript can be used for many things, one of which is directly editing and changing the elements inside of the web page. Maybe you want to completely change the h1 tag, 
	mayble you want certain text to only appear if you click on a button, chances are is that javascript can do it for you. you can link a page to a js file using this code
	inside the head tag:
	<script src="[insert link to file here]"></script>

	In order to get a handle(attach) on to an element, the element MUST have an ID assigned to it. Like CSS, it's best to store javascript into a seperate file, this time it's
	called a .js file. A easy way to test this out is by starting the js file with window.addEventListener('load', function(){}); as this will make the code automatically
	execute once the page loads, this can also be used to load up code if you plan to make buttons change the code. You can also use javascript to make things interactable,
	like an image changes to another image if clicked on or it display an alert upon click.

	Below is a code example of handling a div tag with nothing in it, this assumes the page is linked to the js file. The first line of code tells the page to execute the code 
	inside of it once it loads, if the code requires further attention such as a click, it just loads the code into ram to be executed later. The second line of code gets a
	handle on the div tag with an ID of "hiddenDiv". The third line inserts a h5 element with "Hello" inside of it. The fourth line of code applies a background color while
	the fifth applies a font size.
					
	window.addEventListener('load', function(){

	var div = document.getElementById("hiddenDiv");
	div.innerHTML = "<h5>hello</h5>";
	div.style.backgroundColor = "red";
	div.style.fontSize = "40px";

	});
